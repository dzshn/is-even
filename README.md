# is-even

Return `True` if the given number is even.

## Install

Install with pip:

```sh
$ pip install git+https://github.com/dzshn/is-even
```

From source using [poetry](https://python-poetry.org):

```sh
$ poetry install
```

## Usage

```py
>>> import is_even
>>> is_even(1)
False
>>> is_even(4)
True
```
